# Venda de Vinhos

## Building and Running

---

### Install Dependencies

```sh
npm install
```

### Build project

```sh
npm run build
```

### Start server

```sh
npm run start
```

### Testing

```sh
npm run test
```

---

## Endpoints

- `GET` `/clients/by-purchase-value`
- `GET` `/clients/by-fidelity`
- `GET` `/clients/biggest-purchase-2016`
- `GET` `/recommend/{client_id}`
